π0.5

π0.5 机构：Physical Intelligence 年份：2025 1. 概览 π0.5 是在 π0 基础上的进一步扩展，重点强化了以下几个方面： 长时序任务规划 对新环境的泛化能力 更快且更结构化的预训练方式 高层 / 低层任务分解 它的核心思想是： 对于复杂机器人任务，不能只依赖低层控制。 模型应该先预测一个 高层子任务（subtask），然后在这个子任务条件下生成低层动作。 2. 核心思想 π0.5 的几个核心点包括： 提升 长时序任务 的表现 提升在 新家庭 / 新场景 中的泛化能力 将行为分解为： 高层子任务预测 低层动作生成 使用 FAST tokenization 加速预训练 使用 两阶段训练流程 简而言之，π0.5 从一个纯连续动作的 VLA 模型，演化为一个更结构化的系统，同时结合了： 语言生成 动作 token 预训练 连续动作 flow matching 层级式任务规划 3. 模型结构 3.1 两阶段输出 π0.5 的一个核心结构设计是 两阶段输出机制。 模型首先预测： 高层子任务 / 高层计划 然后再预测： 低层机器人动作 这种设计让它更适合处理长时序任务，因为这类任务通常需要显式的子任务结构。 3.2 多模态输入 token π0.5 使用 (N) 个多模态输入 token。 ...
