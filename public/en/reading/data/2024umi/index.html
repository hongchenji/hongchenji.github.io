  • 核心机制/模型结构 / Key mechanism/model architecture:  

    - hardware interface

        - a parallel jaw gripper with soft fingers

            - trigger-activated

            - handheld

            - 3D printed

        - a wrist mounted GoPro camera as the only sensor

        - side mirror with digital reflection

        - go pro builtin IMU

            - accelerometer

            - gyroscope

            - for slam

                - orb-slam3

            - important for dual arm

        - continuous gripper control

            - fiducial markers

        - kinematic based data filtering

            - filtering according to robot kinematics and dynamics feasibility

            - train only on filtered dataset

    - policy interface

        - Inference-time latency matching

            - Observation latency matching

                - single arm

                    - down sample camera

                    - linearly interpolate gripper and robot proprioception streams

                - dual arm

                    - soft synchronize nearest neigbour frame

                    - max 1/60 seconds off

            - Action latency matching

                - t obs – t input – t output – t act

                - discard all action after t act

                - l camera = t recv - t display - l display

                - l obs = t recv - t robot or 21 x Ping RTT

                - l e2e  manual

        - relative end-effector pose

            - Relative EE trajectory as action representation

                - relative position compare to t0

            - Relative EE trajectory as proprioception

                - position in past 2 frames to provide velocity

            - Relative inter-gripper proprioception

                - relative position between two grippers

        - slam

            - map then localize

            - all data will be relocalize into the map

    - diffusion policy

        - vision encoder

            - Clip pretrained ViT

            - you can choose between ViT-b or ViT-L
