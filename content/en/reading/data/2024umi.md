            - accelerometer

            - gyroscope

            - for slam

                - orb-slam3

            - fiducial markers

        - kinematic based data filtering

            - filtering according to robot kinematics and dynamics feasibility

            - train only on filtered dataset

    - policy interface

        - Inference-time latency matching

            - Observation latency matching

                - single arm

                    - max 1/60 seconds off

            - Action latency matching

                - t obs -- t input -- t output -- t act

                - discard all action after t act

                - l obs = t recv - t robot or 21 x Ping RTT

                - l e2e  manual

        - relative end-effector pose

            - Relative EE trajectory as action representation

                - relative position compare to t0

            - Relative EE trajectory as proprioception

                - position in past 2 frames to provide velocity

                - relative position between two grippers

            - map then localize
